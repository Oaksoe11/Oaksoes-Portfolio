# OakSoe's Portfolio

Personal portfolio for OakSoe Khant, a Software Systems student at Simon Fraser University focused on full-stack web development, AI-powered tools, and secure software practices.

[Live site](https://oaksoes-portfolio.vercel.app/#top)

## About

This site highlights my projects, technical skills, resume, and contact links. It is built as a simple, responsive portfolio with reusable React components and Tailwind CSS styling.

## Featured Projects

- **AI Study Notes Tracker** - Full-stack study tool using Next.js, TypeScript, FastAPI, Supabase, and the Gemini API. [Live](https://ai-study-notes-tracker.vercel.app/) / [Code](https://github.com/Oaksoe11/AI_StudyNotes_Tracker.git)
- **Love At First Paw** - Pet adoption platform built with React, Next.js, TypeScript, the Petfinder API, and the Mapbox API.
- **Portfolio Website** - This site, built with Next.js and deployed on Vercel.

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

Build for production:

```bash
npm run build
```
